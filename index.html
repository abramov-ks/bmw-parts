ИНСТРУКЦИЯ
ШАГ 1 — GOOGLE SHEETS
Открой sheets.google.com → создай новую таблицу → назови её как угодно.
ШАГ 2 — APPS SCRIPT
В таблице: Расширения → Apps Script → удали весь код → вставь содержимое файла Code.gs из архива.
ШАГ 3 — DEPLOY
В Apps Script: Развернуть → Новое развёртывание
Тип: Веб-приложение
Выполнять как: Я
Доступ: Все
→ Нажать Развернуть → разрешить доступ → скопировать URL.
ШАГ 4 — ВСТАВИТЬ URL
Вставь скопированный URL в поле вверху страницы → нажать Сохранить.
ШАГ 5 — GITHUB PAGES
Создай репозиторий на GitHub → загрузи index.html → Settings → Pages → выбрать ветку main → сохранить.
Твоя ссылка: username.github.io/repo-name
